# My PT AI

식단과 운동 기록을 바탕으로 개인화된 AI 코칭을 제공하는 Spring Boot 웹앱입니다.

## Tech Stack

- Java 21
- Spring Boot 3.x
- Spring Web
- Spring Data JPA
- Bean Validation
- Thymeleaf
- Gradle
- H2 Database for local development
- MySQL for production
- Flyway for database migration
- OpenAI API integration from backend only

## Run

```bash
./gradlew bootRun
```

현재 프로젝트는 실무 커밋 단위로 작게 쌓아갑니다.
